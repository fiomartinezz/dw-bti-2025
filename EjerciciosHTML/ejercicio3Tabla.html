ETIQUETAS HMTL
Categoría Etiqueta Nombre o Significado Descripción Principales Usos
Contenedores <div> División Define una sección o contenedor genérico. Agrupa elementos para aplicar estilos o scripts.
<span> Span Define un contenedor en línea genérico. Aplica estilos o scripts a fragmentos de página.
Estructura del documento <body> Cuerpo del documento Contiene el contenido visible de la página web. Muestra el contenido principal de la página.
<html> HTML Elemento raíz de un documento HTML. Contiene todo el contenido de la página web.
Formularios <button> Botón Define un botón interactivo. Crea botones para enviar formularios, realizar acciones, etc.
<form> Formulario Define un formulario para la entrada de datos. Crea formularios para recopilar información del usuario.
<input> Entrada de formulario Define diferentes tipos de campos de entrada. Crea formularios para recopilar información del usuario.
Listas <li> Elemento de lista Define un elemento dentro de una lista. Representa un elemento individual en una lista.
<ol> Lista ordenada Define una lista de elementos con orden específico. Crea listas numeradas.
<ul> Lista no ordenada Define una lista de elementos sin orden específico. Crea listas con viñetas.
Metadatos <!DOCTYPE> Declaración de tipo de documento Define el tipo de documento HTML. Indica la versión de HTML que se está utilizando.
<head> Encabezado del documento Contiene metadatos e información sobre el documento. Define el título de la página, enlaces a hojas de estilo, etc.
<link> Enlace a recursos externos Enlaza a hojas de estilo CSS, iconos, etc. Conecta archivos externos a la página web.
<meta> Metadatos Proporciona metadatos sobre el documento HTML. Define la codificación de caracteres, palabras clave, descripción, etc.
<script> Script de clientes Inserta o enlaza scripts de JavaScript. Añade interactividad a la página web.
<style> Estilos CSS internos Define estilos CSS directamente en el documento HTML. Aplica estilos a los elementos HTML.
<title> Título del documento Define el título de la página que se muestra en la barra de título del navegador. Proporciona un título para la página web.
Multimedia <audio> Audio Inserta un reproductor de audio. Reproduce audio en la página web.
<canvas> Lienzo Define un área para gráficos dinámicos. Crea gráficos y animaciones con JavaScript.
<iframe> Marco en línea Inserta otro documento HTML dentro del documento actual. Incrusta contenido de otras páginas web.
<img> Imagen Inserta una imagen en la página. Muestra imágenes en la página web.
<svg> Gráfico vectorial escalable Inserta gráficos vectoriales. Muestra gráficos vectoriales escalables.
<video> Video Inserta un reproductor de video. Muestra videos en la página web.
Secciones <article> Artículo Define un artículo independiente. Representa contenido independiente como un artículo de blog.
<aside> Contenido secundario Define contenido secundario o complementario. Muestra barras laterales, anuncios, etc.
<footer> Pie de página Define el pie de página de una sección o página. Contiene información de contacto, enlaces de pie de página, etc.
<header> Encabezado de sección Define el encabezado de una sección o página. Contiene información introductora o de navegación.
<main> Contenido principal Define el contenido principal de la página. Contiene el contenido central de la página.
<nav> Navegación Define una sección de enlaces de navegación. Crea menús de navegación.
<section Sección Define una sección genérica dentro de un documento. Agrupa contenido relacionado temáticamente.
Tablas <table> Tabla Define una tabla de datos. Muestra datos en filas y columnas.
<td> Celda de tabla Define una celda de datos dentro de una fila de tabla. Contiene datos individuales en una tabla.
<th> Celda de encabezado de tabla Define una celda de encabezado dentro de una fila de tabla. Contiene encabezados de columna o fila.
<tr> Fila de tabla Define una fila dentro de una tabla. Agrupa celdas horizontalmente.
Texto <a> Ancla Crea un hipervículo a otra página o recurso. Enlaza a otras páginas web, archivos, etc.
<br> Salto de línea Inserta un salto de línea. Crea saltos de línea en el texto.
<h1> a <6> Encabezados Definen encabezados de diferentes niveles. Estructuran el contenido en secciones jerárquicas.
<hr> Línea horizontal Inserta una línea horizontal. Separa secciones de contenido.
<p> Párrafo Define un párrafo de texto. Agrupa texto en párrafos.
