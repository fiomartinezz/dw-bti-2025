ETIQUETAS DE FORMULARIO EN HTML
Etiqueta Descripción Tipo
<form> Etiqueta contenedora que define un formulario Bloque
<label> Indica la descipción de un campo Línea
<input> Define un campo de entrada, acompañado del atributo type Línea
<select> Define una lista desplegable Línea
<option> Define una opción de la lista desplegable Línea
<textarea> Define un campo de varias líneas de texto Línea
<button> Define un botón con función propia Línea
<fieldset> Permite agrupar los campos de un formulario Bloque
<legend> Crea un título para un grupo de campos Línea
